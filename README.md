## 🎮 Overview

**Shadow Kombat: Cyber Nexus Online** is a fully self-contained, AAA-quality 2D fighting game built in pure **HTML5, CSS3, and Vanilla JavaScript** — no frameworks, no build tools, no dependencies. Drop a single `.html` file and you're fighting.

Inspired by the golden era of arcade fighting games, Shadow Kombat brings modern-day polish: **real-time online PvP via Socket.IO**, a **cinematic result screen**, **smart camera system**, **procedurally animated fighters**, **dynamic particle FX**, and a deep **coin/XP economy** — all rendering at a silky 60 FPS with adaptive performance scaling.

```
Zero dependencies · Zero build steps · Single HTML file · ~3,500 lines of pure craft
```

## ✨ Features

### ⚔️ Core Combat
- **12 unique fighters** across 4 rarity tiers — Starter, Rare, Legendary, Mythic
- **Full moveset per fighter** — Punch, Kick, Grab, 3 Specials, X-Ray, Rage Mode, Fatality
- **Parry system** — frame-perfect parries grant invincibility and bonus damage
- **Counter hits** — punish enemies mid-animation for 35% bonus damage
- **Combo engine** with scaling damage and live HUD combo counter
- **Freeze, Poison, Stagger** — character-specific status effects
- **Hitbox-based collision** with per-frame resolution

### 🌐 Online Multiplayer
- **Quick Match** — auto-matchmaking by rank
- **Ranked Match** — compete on the global leaderboard
- **Friend Match** — private room codes with clipboard copy
- **Spectate Mode** — watch live battles in real time
- **In-game chat** with text and emoji emotes
- **Client-side prediction** with server reconciliation for lag compensation
- **Ping display** with region detection (Optimal / Good / High Latency)
- **Socket.IO** architecture — drop in your server URL to go live

### 🎨 Visuals & Effects
- **Procedurally animated 3D-rig fighters** — full body with limb IK, armor layers, glows
- **Smart cinematic camera** — dynamic zoom, tracking, and punch-in on heavy hits
- **6 fully realized arenas** — Inferno Temple, Frozen Tundra, Shadow Nexus, Thunder Peak, Cyber Palace, Void Realm
- **Weather systems** — rain, snow, lightning, fire embers per arena
- **Particle engine** with burst, shockwave, sparks, and floating damage numbers
- **Hit stop frames** scaling with damage (1–6 frames)
- **Chromatic aberration**, **slow motion**, and **screen flash** on X-Ray moves
- **Crowd system** with excitement reactions to big hits
- **Holographic scanlines**, **RGB border**, **cyber grid** background effects
- **Cinematic letterbox result screen** with particle rain

### 🏆 Progression & Economy
- **Coin economy** — earn by winning; harder difficulty = more coins
- **XP & Level system** — 10 levels with named tiers (Rookie → Grand Master)
- **Rank Points** — separate online rank from Bronze to Cyber Legend
- **Upgrade tree** — Punch Power, Kick Power, Move Speed, Defense (5 levels each)
- **Shop** — buy locked fighters, cosmetic skins, and ability upgrades
- **10 cosmetic skins** including Rainbow, Neon, Void, Mythic (animated)
- **Achievements** — First Win, Veteran Fighter, Brutal Finisher, Combo Destroyer, and more
- **Persistent save** via `localStorage`

### 🔊 Audio
- **Procedural audio engine** using the Web Audio API — zero audio files required
- Unique sounds for: punch, kick, special, X-Ray, grab, block, parry, jump, KO, fatality, win, rage, coins
- **Dynamic background music** with kick drum, hi-hat, and bass patterns at 155 BPM
- All audio generated programmatically with oscillators, noise buffers, and filters

## 🌐 Online Multiplayer

Shadow Kombat uses a **Socket.IO** client architecture. The game ships in **Demo Mode** (solo/AI fallback) when no server URL is configured.

### To Enable Real Online Play

1. Deploy a Socket.IO server (Node.js recommended — see below)
2. Open the HTML file and locate this line near the top of the `<script>`:
```js
const SOCKET_URL = '';  // Leave empty for local dev (will use simulated mode)
```
3. Replace with your deployed server URL:
```js
const SOCKET_URL = '(https://yasirshaikh03.github.io/shadow-kombat/)';
```

### Required Socket.IO Server Events

| Event (Client → Server) | Payload |
|---|---|
| `setUsername` | `{ username, rankPoints, handle }` |
| `quickMatch` | `{ rankPoints, username, handle }` |
| `rankedMatch` | `{ rankPoints, username }` |
| `createRoom` | `{ username, rankPoints }` |
| `joinRoom` | `{ code, username, rankPoints }` |
| `startRoomMatch` | `{ code }` |
| `inputSync` | `{ seq, vx, jump, punch, kick, ... }` |
| `chatMessage` | `{ msg, username, isEmote }` |
| `ping_custom` | *(empty)* |
| `spectate` | *(empty)* |

| Event (Server → Client) | Payload |
|---|---|
| `playerCount` | `{ count }` |
| `roomCreated` | `{ code }` |
| `playerJoined` | `{ id, username, emoji }` |
| `matchFound` | `{ opponentId, p1Char, p2Char }` |
| `matchStart` | `{ p1Char, p2Char }` |
| `inputSync` | `{ vx, jump, punch, kick, ... }` |
| `stateSync` | `{ p1hp, p2hp, p1x, p2x, frame }` |
| `chatMessage` | `{ username, msg, isEmote }` |
| `pong_custom` | *(empty)* |

## ☁️ Deployment

### Static Hosting (Frontend only, Demo Mode)
Deploy to **GitHub Pages**, **Netlify**, **Vercel**, or **Cloudflare Pages** — no build step required.

```bash
# GitHub Pages: push index.html to /docs or root
# Netlify: drag and drop the file or connect your repo
```

### Full Online PvP (Socket.IO Server)

A minimal Node.js server example:

```js
const express = require('express');
const http = require('http');
const { Server } = require('socket.io');

const app = express();
const server = http.createServer(app);
const io = new Server(server, { cors: { origin: '*' } });

io.on('connection', (socket) => {
  socket.on('ping_custom', () => socket.emit('pong_custom'));
  socket.on('setUsername', (data) => { socket.username = data.username; });
  // Add your matchmaking, room, and sync logic here
});

server.listen(3000, () => console.log('Shadow Kombat server on port 3000'));
```

Deploy your server to **Render**, **Railway**, **Fly.io**, or **Heroku**, then update `SOCKET_URL` in the HTML.

| Tier | Target FPS | Max Particles | Rain | Trails | Crowd |
|---|---|---|---|---|---|

| 🥇 HIGH | 50+ | 95 | ✅ | ✅ | ❌ |
| 🥈 MEDIUM | 40+ | 55 | ❌ | ❌ | ❌ |
